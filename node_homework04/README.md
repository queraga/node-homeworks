## Node.js Homework - Moment & File Operations

# Task 1 - Date Formatting with Moment

In this task, we:
• Initialized a Node.js project
• Installed the moment library
• Retrieved the current date and time
• Formatted the date in three formats:
• DD-MM-YYYY
• MMM Do YY
• dddd
• Printed the formatted results to the console

Purpose:
To practice working with external npm packages and date formatting.

# Task 2 - File Creation & Reading using fs and dotenv

In this task, we:
• Installed and configured dotenv
• Created a .env file with: FILENAME=myfile.txt
• Loaded environment variables using dotenv.config()
• Created a file dynamically using the filename from process.env
• Wrote text into the file using fs.writeFile
• Read the file content using fs.readFile
• Printed the file content to the console

Purpose:
To understand:
• Environment variables
• Asynchronous file operations
• Callback handling in Node.js
• Proper error handling

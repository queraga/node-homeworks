# Homework 7 - Sequelize Setup and Migration

### Description

This project demonstrates how to configure Sequelize ORM in a Node.js application and manage a MySQL database using models and migrations.

### Tech Stack

- Node.js
- Sequelize
- Sequelize CLI
- MySQL
- mysql2
- dotenv

### What Was Implemented

- Initialized a Node.js project
- Installed and configured Sequelize ORM
- Connected the application to a MySQL database
- Created a Sequelize model App
- Generated and configured a migration
- Applied the migration to create the Apps table in the database
- Secured the database password using .env

### What I Learned

- How to configure Sequelize in a Node.js project
- How to connect a backend application to MySQL
- How to define Sequelize models
- How to create and run migrations
- How to manage database schema changes using Sequelize CLI
